## Lung Cancer Risk Prediction (Research Project)  
Technical University of Munich

This research project focuses on predicting lung cancer risk over a six-year period using longitudinal medical data and deep learning models.

The system combines temporal data with anatomical features to identify long-term patterns and improve early cancer risk prediction.

### Key technical elements
- Python
- PyTorch
- Transformer-based models
- Medical imaging
- Deep learning
- Data analysis

### Contributions
- Developed and tested machine learning models
- Conducted experiments and model evaluations
- Worked with real-world clinical datasets
- Visualized model outputs and performance

### Outcome
- Project is being prepared for scientific publication
- Demonstrates strong research-level AI development skills
